Drawing Retrieval System

Drawing retrieval

By selecting a sequence of filters the system drills down to a small set of drawings that you are looking for.

Each drawing's captured information is displayed with a reasonable size image of the drawing which has sufficient detail for you to determine which drawing is actually being searched.

By selecting the image, a high resolution PDF drawing is opened in a new session for more detailed analysis.
